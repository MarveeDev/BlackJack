
# BlackJack

Complex singleplayer BlackJack python game made by using customtkinter package


## Installation

Install my-project with npm

```bash
  cd your-directory
  git clone https://pietroorg/blackjack
  pip install customtkinter
  pip install pillow
  pip install pymongo
  python3 main.py
```
